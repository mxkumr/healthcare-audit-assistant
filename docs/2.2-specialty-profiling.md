# 2.2 Specialty Peer Profiling

App: `app/2.2aspecialty-profiling/` · Data: CMS Medicare Physician & Other Practitioners (`medicare-ProviderSummary.csv`)

Builds on **Task 2.1** (`ProviderCostEfficiency`) — same per-provider cost and utilization metrics, now compared to **specialty peers** instead of fixed dollar cutoffs.

---

## What we're doing

We ask: *"How abnormal is this provider **compared to others in the same specialty**?"*

**2.1** uses fixed thresholds ($900/patient, 15 services). **2.2** computes a **national peer baseline** per specialty-year, then flags providers who deviate far above or below that average.

**Example:** Ambulance provider **ABOVE AVERAGE EMS LLC** (Texas, 2019) bills **$20,435.90 per patient**. That sounds like fraud on a spreadsheet — but the ambulance specialty average is only **$1,549.30**. This provider is **+1,219% above peers**, the worst cost outlier in their specialty. Task 2.2 surfaces that relative signal; pair with Task 2.1 drill-down for clinical context (tiny panel, 11.10 risk score, ambulance mileage units).

**One row in the app** = one provider × year with peer baseline joined (e.g. 2019 · NPI 1003245093 · Ambulance Service Provider).

---

## What you see

| View | Shows |
|------|-------|
| **Chart (top)** | Horizontal bar — specialty vs **max** cost peer deviation (%) |
| **Table (bottom)** | Providers grouped by **Specialty**; expand to see individuals |

**Filters:** Year (defaults to **2022**), Specialty, State

**Example — reading the chart:**

Each bar is one CMS specialty. Bar length = **worst cost outlier** in that specialty (MAX `CostTierDeviation`). Ambulance Service Provider bars are long because one provider can be +1,000% above peers even when the specialty average is modest.

**Tip:** Always filter to a **single year**. Baselines are computed per year — mixing years blends unrelated peer averages.

---

## How the math works

### Step 1 — National peer baseline (`SpecialtyPeerBaselines`)

One row per **Year + Specialty**:

```
National Avg Cost     = avg(Cost per Patient) across all providers in that specialty-year
National Avg Services = avg(Services per Patient) across all providers in that specialty-year
```

**Example — Ambulance Service Provider, 2019:**

```
National Avg Cost = $1,549.30 per patient
```

Every ambulance provider in 2019 shares this same baseline number.

---

### Step 2 — Provider deviation (`SpecialtyPeerDeviations`)

Each provider is joined to their specialty-year baseline:

```
Cost per Patient     = Tot_Mdcr_Pymt_Amt ÷ Tot_Benes
Services per Patient = round(Tot_Srvcs ÷ Tot_Benes)

Cost Tier Deviation (%) = round(((Cost per Patient − National Avg Cost) ÷ National Avg Cost) × 100)

Service Tier Deviation (%) = round(((Services per Patient − National Avg Services) ÷ National Avg Services) × 100)
```

Division uses `nullif(baseline, 0)` — zero baselines return **0%** instead of crashing.

| Deviation | Meaning |
|-----------|---------|
| **Positive (+)** | Provider bills **more** than specialty peers |
| **Negative (−)** | Provider bills **less** than specialty peers |
| **Near 0** | In line with specialty average |

---

## UI signal thresholds (criticality colors)

Deviation columns use **percentage-based** traffic lighting — not the fixed dollar bands from 2.1.

| Zone | Cost / Service Deviation | Color signal |
|------|--------------------------|--------------|
| **On target** | 0 – 25% above peers | Green |
| **Elevated** | 25 – 100% above peers | Yellow |
| **Critical** | > 100% above peers | Red |

Direction is **minimize** — higher positive deviation = worse audit signal.

These thresholds apply to **Cost Peer Deviation (%)** and **Service Peer Deviation (%)** in the table and chart data points.

---

## 2.1 vs 2.2 — when to use which

| Question | Use |
|----------|-----|
| "Is this provider above $900/patient regardless of specialty?" | **2.1** fixed Cost Classification |
| "Is this provider abnormal **for an ambulance provider**?" | **2.2** peer deviation |
| "Which specialty has the worst outliers?" | **2.2** chart (MAX per specialty) |
| "Why might a +1,219% outlier be legitimate?" | **2.1** object page (risk score, comorbidities) |

## Columns

| Column | What it means | How it's calculated |
|--------|---------------|---------------------|
| **Specialty** | CMS provider type | `Rndrng_Prvdr_Type`; table groups by this field |
| **Provider Name** | Organization or clinician | CMS name, uppercased |
| **NPI** | National Provider Identifier | CMS `Rndrng_NPI` |
| **Cost per Patient** | This provider's spend intensity | `Tot_Mdcr_Pymt_Amt ÷ Tot_Benes`; collapsed groups show **MAX** (peak outlier) |
| **National Avg Cost** | Specialty-year peer baseline | `avg(Cost per Patient)` from `SpecialtyPeerBaselines` |
| **Cost Peer Deviation (%)** | % above/below peer cost average | `round(((Cost − Baseline) ÷ Baseline) × 100)` |
| **Service Peer Deviation (%)** | % above/below peer service average | `round(((Services − Baseline) ÷ Baseline) × 100)` |

**Hidden from table but in OData:** State, Services per Patient, National Avg Services (available for filters and future drill-down).

## Worked example — ABOVE AVERAGE EMS LLC (2019)

Raw joined row from the dataset:

| Field | Value |
|-------|-------|
| Specialty | Ambulance Service Provider |
| Cost per Patient | $20,435.90 |
| National Avg Cost | $1,549.30 |
| Cost Peer Deviation | **+1,219%** |
| Service Peer Deviation | **+2,043%** |

**Cost deviation math:**

```
((20,435.90 − 1,549.30) ÷ 1,549.30) × 100 = 1,219%  →  Critical (red)
```

**Why 2.2 matters here:** In absolute dollars this provider is extreme. Against **ambulance peers only**, they are still extreme — but now you know the *relative* scale. A +1,219% signal means "12× the specialty norm," not just "above $900."

---

## Contrast example — Internal Medicine (2019)

Same logic, different economics:

| Provider | Cost/Patient | National Avg | Deviation |
|----------|-------------|--------------|-----------|
| LEATHERWOOD | $1,629.84 | $254.67 | **+540%** |

Internal Medicine's national average is only **$255/patient**. A provider at $1,630 is nowhere near ambulance-level dollars — but still **+540% above Internal Medicine peers**. That is a strong relative outlier worth triaging.

**Takeaway:** 2.2 compares apples to apples within each specialty. Cross-specialty dollar comparisons are misleading.

---

## Reading grouped table rows

The table **groups by Specialty**. Collapsed parent rows aggregate differently from expanded child rows:

| Row level | Cost per Patient | National Avg Cost |
|-----------|------------------|-------------------|
| **Collapsed specialty group** | **MAX** — worst outlier in specialty | **MIN** — peer baseline (same for all) |
| **Expanded provider row** | Individual provider cost | Same specialty baseline |

If both columns looked identical on a collapsed row before, that was because both were averaged — now parent rows show **peak outlier vs baseline** side by side.
